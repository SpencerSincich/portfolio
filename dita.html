Project description



Flipp is an app that helps people save money on shopping.



I created the online help instructions by using the app: Oxygen XML Editor.



Oxygen enables the writer to use the DITA standard to create documents that include text and images.



One of the main features of the DITA standard is single-sourcing. This enables the writer to quickly reference a section of source content to display in several output sections.



To single-source the "Copyright" section in this online help project, I used the XML attribute called: “Content Reference”, <conref>.



In large projects where a lot of content is being reused, single-sourcing can save time.
