Project description



The Customer Reviews plugin enables the ecommerce merchant to configure the processes related to customers submitting product reviews.



    The excerpts from the user manual include:
  • Diagrams that provide an overview of the review process.
  • Instructions for how to configure a section of the settings.
  • Explanations about the configuration changes.
  • Instructions for how to import reviews from an Excel template.
