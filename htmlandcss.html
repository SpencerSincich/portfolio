Note: This independent portfolio project is not connected to the company: Xpenditure (now called Rydoo). I don't own any of the rights related to the Xpenditure app. 2020.



Project description



Xpenditure was an expense reporting app that enabled employees to submit their work expenses to the accounting department of their company.



I created the portfolio project to improve my ability to use HTML and CSS to hand-code websites.



    The user interface includes these features:
  • Responsive layout (4-breakpoints).
  • Hamburger menu that uses a reference to the jQuery JavaScript library.
  • Meta data (e.g. keywords, description).
  • Hyperlinks that transfer the site to new pages.
  • PNG images that hyperlink to the company's social media sites.
  • Ordered lists.
  • Unordered lists.
  • Emphasis tags (e.g. strong).
  • Span tags.
  • Breadcrumbs.
  • Screenshot images edited in Photoshop.
  • Favicon.
  • Font families.
  • Colors (using both name and hexadecimal formats).
  • Classes and IDs.
  • Styling referenced to a single CSS page.


Project software



    I created the user interface by using:
  • Sublime Text (source code editor).
  • Adobe Photoshop CC (image editor).
