Project description



For this freelance project, I drafted KB articles to help the users of a mapping app.


Quick start guide - (Demo app)


The sample articles teach the map creator how to build custom filters for overlays on a map. For example, real estate listings in New York City. The map viewer can then use the filters to segment which overlays are displayed.



    I created the sample articles by:
  • Interviewing the developer.
  • Reviewing the video and testing the app.
  • Writing definitions, instructions, and simple examples.


Project software



    I created the sample articles by using:
  • Google Meet (video conferencing platform).
  • TechSmith Camtasia (screencast recorder).
  • Google Docs (document editor).
  • Sublime Text (source code editor).
