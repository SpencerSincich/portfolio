Introduction

The Teams section enables you to specify the role-based permissions each user has.

The user roles are:
  • Owner.
  • Admin.
  • Regular.
Note: An Admin or Owner can access all projects.

View Contents ↑

 

Owner

How many Owners per company? 1
Can access all projects? Yes
Can create projects? Yes
Can create custom color categories for tasks? Yes
Can access the subscribe page for payment info? Yes
Can select Admins for company? Yes
Can archive/unarchive users? Yes
Is automatically added to every project, and can't be removed? Yes
Is added/removed from a project by an Admin or Owner? N/A

View Contents ↑

 

Admin

How many Admins per company? Unlimited
Can access all projects? Yes
Can create projects? Yes
Can create custom color categories for tasks? Yes
Can access the subscribe page for payment info? No
Can select Admins for company? Yes
Can archive/unarchive users? Yes
Is automatically added to every project, and can't be removed? No
Is added/removed from a project by an Admin or Owner? Yes
