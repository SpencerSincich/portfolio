How to manually add an attribute to an overlay

  • May 08, 2022 22:04
  • Updated
Note: To save attributes, you must be logged into a Pro account. To learn how to upgrade, visit this link: How to upgrade to a Pro account

Definition and examples of attributes

An overlay’s attributes are listed data describing an overlay. The users who edit the map add the listed data to describe the overlay to map viewers.

For example: A marker overlay represents a house for sale. So the marker's attributes describe the house.
 
Note: To view the attributes, the map viewers click on the overlay in the Map View.
 
Each attribute is a name-value pair, including an:
  • Attribute-name.
    AND
  • Attribute-value.
For example, the house (marker overlay) has these 4 attributes:
 
Name Value
PROPERTY TYPE Single Family Residential
ADDRESS 10393 Abrams Dr
CITY Colorado Springs
PRICE $100,000
 

An attribute-name can be a shared category for multiple overlays.

Create the overlay

  1. Log into Example.com at:
    https://www.example.com/create 
     
  2. In the Map View, create an overlay. For example: a marker.
    Visit this link to learn how to:
    Create a marker - (Marker Tool) 
     

Manually add an attribute to the overlay

  1. Select the Edit/Measure Tool in the top toolbar.
    Or you can press the "Q" button on your keyboard.
  2. In the Map View, right-click on the overlay.
     
    In the menu that appears, click: "Edit Attributes".
    The "ATTRIBUTES" window will appear.
  3. Copy the attribute’s name.
    For example: "PROPERTY TYPE".
    Copy by clicking-and-dragging to highlight the text inside the above example’s quotation marks. Then use your keyboard to press:
    • Ctrl + C (for Windows).
    • Command + C (for Macs).
     
  4. Paste the attribute's name.
    Paste by clicking on the empty field under the "Attribute" heading in the attributes window. Then use your keyboard to press:
    • Ctrl + V (for Windows).
    • Command + V (for Macs).
     
     
  5. Copy the attribute's value.
    For example: "Single Family Residential".
  6. Paste the value under the "Value" heading in the attributes window.
  7. (Optional) If you want to automatically add the attribute-name to other overlays, click the "Shared" checkbox on the left-side of the attribute's row.
    For users who edit the map:
    The shared attribute-name will be an optional row in the attributes window for all overlays.
     
    For map viewers who click on an overlay:
    The shared attribute-name will only be visible if a map editor added an attribute-value.
  8. (Optional) If you clicked the "Shared" checkbox.
     
    Under the "Type" heading on the right-side of the attribute's row, click to select the default: "ANY".
    By selecting "ANY", you enable the attribute-value to be any type of data. For example: text, or numbers.
     
    To learn how to enforce the type of attribute-values, visit this link: Enforce the type of values in a shared attribute
  9. (Optional) If you want to add another attribute to the overlay, click the button: "+ Add Attribute".
    The button is located under the attribute rows in the attributes window.
  10. (Optional) If you want to delete an attribute, click the Garbage Bin Icon on the attribute’s row.
  11. When complete, click the button: "SAVE ATTRIBUTES".
  12. (Optional) In the overlay editor window, click the "SAVE" button.

View and copy the data in the info bubble

For each type of overlay (not including labels), map users can quickly view the overlay's attributes in the info bubble.
  1. Select the Hand Tool in the top toolbar.
    Or you can hold down the "Shift" button on your keyboard.
  2. In the Map View, click on the overlay. For example: a marker.
  3. You will see the info bubble.
  4. (Optional). If you want to copy the data in an attribute’s field, click on a name or value. The data will be highlighted. Copy the data by pressing these keyboard buttons:
    • Ctrl + C (for Windows).
    • Command + C (for Macs).
