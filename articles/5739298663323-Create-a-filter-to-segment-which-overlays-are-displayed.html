Create a filter to segment which overlays are displayed

  • May 07, 2022 21:00
  • Updated
Note: To create a filter, you must be logged into a Pro account. To learn how to upgrade, visit this link: How to upgrade to a Pro account
 

Create a filter to segment which overlays are displayed

  • (Step 1)
    Manually add an attribute to an overlay
     
  • (Step 2)
    Create a single condition in a single filter
     
  • (Step 3)
    Create multiple conditions in a single filter
 

(Step 1) - Manually add an attribute to an overlay

Perform the procedure in the article at this link: How to manually add an attribute to an overlay
 
 

(Step 2) - Create a single condition in a single filter

Definitions

A filter is a set of conditions you configure to block/allow the display of overlays.

A condition is a rule you configure to block/allow the passing of overlays through a filter.

You configure each condition (rule) by considering the overlay attributes.
 
 

Add the first attribute row to 3 markers

In this example, a real estate buyer views a map displaying houses for sale. The buyer wants to apply a filter so the map only displays the houses priced between: $100,000 and $200,000.

  1. If you aren't logged into Example.com, log into your account at:
    https://www.example.com/create 
  2. (Optional) If the map contains overlays, create a new map.
    To create a new map:
    1. Click the "MENU" button in the top toolbar.
       
    2. Under the "Example.com" heading, click the button: "New Map".
       
    3. (Optional) Click: "Yes".
       
    4. Click: "Ok".
       
    5. In the top-right corner of the menu window, click the "X" button.
       
    6. In the Map View, you will see the new map.
     
  3. Create 3 markers including these attributes:
    Marker 1
    Name Value
    PRICE $99,000
     
    Marker 2
    Name Value
    PRICE $150,000
     
    Marker 3
    Name Value
    PRICE $199,000
 
 
 

Create a single condition in a filter

  1. In the top of the Layers Panel, click the Filter Icon displaying the tooltip: "Add Filters".
     
  2. The "FILTERS" window will appear asking you to select a template. For this example, select: "Blank".
     
  3. In the "FILTERS" window, you will see a colored square (example: blue) above the default condition line.
    In later steps, the square's color will appear on the left-side of individual overlay rows in the Layers Panel.
    The color will show which overlays have passed through the filter.
     
  4. On the default condition line, click the arrowhead on the right-side of the first field. In the drop-down, select the attribute you previously created: "PRICE".
     
  5. Click the arrowhead on the right-side of the second field. In the drop-down, select: "IS BETWEEN".
    You will see the 5th and 6th fields added to the condition line.
     
  6. Click the arrowhead on the right-side of the third field. In the drop-down, select: "Value".
     
  7. On the fourth field, copy-and-paste the price of "$100,000".
     
  8. Click the arrowhead on the right-side of the fifth field. In the drop-down, select: "Value".
     
  9. On the sixth field, copy-and-paste the price of "$200,000".
     
  10. At the bottom of the "FILTERS" window, click the button: "SAVE FILTERS".
     
  11. In the Map View, you will see "Marker 1" has been filtered out, because the price, ($99,000), is not between $100,000 and $200,000.
    Only Marker 2 and Marker 3 have passed through the filter.
    Marker 2
    PRICE is between $100,000 and $200,000
     
    Marker 3
    PRICE is between $100,000 and $200,000
     
 
 
 
 

Toggle the condition on/off

  1. In the filters section at the top of the Layers Panel, you will see the checkbox condition line: "PRICE is between $100,000 and $200,000"
     
  2. To toggle the condition on/off, click the box on the left of the checkbox condition line.
    When the checkbox condition line is toggled on, you will see the colored rectangle on the left-side of each overlay row that passes through the filter.
     
    On the checkbox condition line, you will see the filter icon is the same color.
     
  3. To prepare for the next steps, toggle the condition off, so the 3 markers are visible.
 

(Step 3) - Create multiple conditions in a single filter

Add the second attribute row to the 3 markers

The real estate buyer wants the filter to include these types of houses:
"Property type is single family residential".
 
Add these attributes to the 3 markers:
 
Marker 1
Name Value
PROPERTY TYPE Single Family Residential
 
Marker 2
Name Value
PROPERTY TYPE Single Family Residential
 
Marker 3
Name Value
PROPERTY TYPE Townhouse
 

Refresh the filters

In the filters section at the top of the Layers Panel, the Refresh Icon appears after you make changes to the overlays. Click the Refresh Icon to update the filters.
The Refresh Icon is 2 arrows forming a circle.

Create the second condition in a single filter

  1. In the top of the Layers Panel, click the Filter Icon displaying the tooltip: "Add Filters".
     
  2. In the "FILTERS" window, you will see the condition you added in the previous example. Under the first condition, click the button: "+ Add Condition".
     
  3. At the top of the "FILTERS" window, you will see the single filter’s colored square now has a drop-down menu. Click the drop-down to select: "Match ANY conditions".
     
  4. On the new condition line, click the arrowhead on the right-side of the first field. In the drop-down, select the attribute: "PROPERTY TYPE".
     
  5. Click the arrowhead on the right-side of the second field. In the drop-down select: "IS".
     
  6. Click the arrowhead on the right-side of the third field. In the drop-down select: "Unique".
    For your reference:
    • Value - Requires you to input a value.
    • Attribute - Enables you to select from all of the "attribute names" added to overlays.
    • Unique - Enables you to select from all of the "attribute values" added to overlays.
     
     
  7. Click the arrowhead on the right-side of the fourth field. In the drop-down select: "Single Family Residential".
     
  8. At the bottom of the "FILTERS" window, click the button: "SAVE FILTERS".
     
  9. In the filters section at the top of the Layers Panel, click the checkboxes for this configuration:
    Match ANY - on.
     
     
    • PRICE is between $100,000 and $200,000 - on.
    • PROPERTY TYPE is Single Family Residential - on.
    In the Layers Panel, you will see all 3 markers have passed through the filter.
     
 
 
 

Require overlays to "match ANY conditions" to pass through a single filter

  1. In the top of the Layers Panel, click the Filter Icon displaying the tooltip: "Add Filters".
     
  2. At the top of the "FILTERS" window, you will see the single filter’s drop-down menu displays the setting: "Match ANY conditions".
    "Match ANY conditions" means the filter will display the overlays that match any of the multiple conditions.
     
  3. In the top-right corner of the "FILTERS" window, click the "X" button.
     
  4. Above the 2 checkbox condition lines in the filters section (Layers Panel), you will see the checkbox: "Match ANY".
    If you click to toggle "Match ANY" off, the 2 checkbox conditional lines will be toggled off.
     
    If you toggle "Match ANY" on, you can select whether each checkbox condition line is toggled on/off.
     
    Since "Match ANY" is on, you can toggle one of the condition lines off so:
    The overlays only have to pass through the other condition to be displayed.
     
  5. For this example, click the checkboxes to toggle "on" all 3.
    Match ANY - on.
    • PRICE is between $100,000 and $200,000 - on.
    • PROPERTY TYPE is Single Family Residential - on.
     
     
  6. In the Map View, you will see all 3 overlays are displayed, because:
     
    Marker 1 matches any condition
    PROPERTY TYPE is Single family residential
     
    Marker 2 matches any condition
    PRICE is between $100,000 and $200,000
     
    PROPERTY TYPE is Single family residential
     
    Marker 3 matches any condition
    PRICE is between $100,000 and $200,000
 
 
 

Require overlays to "match ALL conditions" to pass through a single filter

  1. In the top of the Layers Panel, click the Filter Icon displaying the tooltip: "Add Filters".
     
  2. At the top of the "FILTERS" window, you will see the single filter’s drop-down menu displays the setting: "Match ANY conditions".
     
  3. Click the single filter’s drop-down menu. Then select the setting: "Match ALL conditions".
    "Match ALL conditions" means the filter will only display the overlays that match all of the multiple conditions.
     
  4. At the bottom of the "FILTERS" window, click the button: "SAVE FILTERS".
     
  5. Above the 2 checkbox condition lines in the filters section (Layers Panel), you will see the checkbox: "Match ALL".
     
  6. For this example, click the checkboxes to toggle "on" all 3.
    Match ALL - on.
    • PRICE is between $100,000 and $200,000 - on.
    • PROPERTY TYPE is Single Family Residential - on.
     
     
  7. In the Map View, you will see only Marker 2 is displayed, because:
     
    Marker 1 doesn't match all conditions
     
    Marker 2 matches all conditions
    PRICE is between $100,000 and $200,000
     
    PROPERTY TYPE is Single family residential
     
     
    Marker 3 doesn't match all conditions
 

(Optional) Add a custom label to a filter's checkbox 

  1. In the top of the Layers Panel, click the Filter Icon displaying the tooltip: "Add Filters".
     
  2. At the top of the "FILTERS" window, you will see the single filter’s drop-down menu displays the setting: "Match ALL conditions".
     
  3. To the right of the drop-down menu, click the field: "Label (optional)".
  4. Type-in the words for the custom label.
    For example: "Price and Property type".
  5. At the bottom of the "FILTERS" window, click the button: "SAVE FILTERS".
  6. In the filters section at the top of the Layers Panel, you will see the filter's checkbox is labeled: "Price and Property type".
    Price and Property type - on.
    • PRICE is between $100,000 and $200,000 - on.
    • PROPERTY TYPE is Single Family Residential - on.
     
 

(Optional) Hide a filter's condition lines

  1. In the top of the Layers Panel, you will see the filter's label. For example: "Price and Property type".
  2. To the right of the checkbox's label, click the arrowhead.
  3. You will see the filter's label continues to be displayed, and the 2 condition lines are hidden from view.
    Price and Property type - on.
