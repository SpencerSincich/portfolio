Enforce the type of values in a shared attribute

  • Updated

Any

"ANY" allows any type of value in the attribute-value field, including: words, and numbers.
 
To select the "ANY" type, the attribute can be set to shared or not shared.
  1. In the overlay's "ATTRIBUTES" window, find or add an empty attribute row.
  2. Under the "Type" heading, click on the drop-down menu. Then select: "ANY".
  3. (Optional) Under the "Shared" heading, click to add a check to the box.
  4. Under the "Attribute" heading, click on the empty field. Then input the attribute-name. For example: "PROPERTY TYPE".
  5. Under the "Value" heading, click on the empty field. Then input the attribute-value. For example: "Single Family Residential".
  6. (Optional) If this is the last attribute you're adding, click the button: "SAVE ATTRIBUTES".

Number Any

"NUMBER ANY" only allows numbers in the attribute-value field.
  1. In the overlay's "ATTRIBUTES" window, find or add an empty attribute row.
  2. Under the "Shared" heading, click to add a check to the box.
  3. Under the "Attribute" heading, click on the empty field. Then input the attribute-name. For example: "YEAR BUILT".
  4. Under the "Type" heading, click on the drop-down menu. Then select: "NUMBER ANY".
  5. Under the "Value" heading, click on the empty field. Then input the attribute-value. For example: "2012".
  6. (Optional) If this is the last attribute you're adding, click the button: "SAVE ATTRIBUTES".

Number Range

"NUMBER RANGE" limits which numbers can be inputted in the attribute-value field. The numbers must be within the range you specify. Optionally, you can also limit possible numbers by setting a step (interval) value.
  1. In the overlay's "ATTRIBUTES" window, find or add an empty attribute row.
  2. Under the "Shared" heading, click to add a check to the box.
  3. Under the "Attribute" heading, click on the empty field. Then input the attribute-name. For example: "ESTIMATED PRICE".
  4. Under the "Type" heading, click on the drop-down menu. Then select: "NUMBER RANGE".
    The "ATTRIBUTE OPTIONS" window will appear.
  5. Input the minimum, maximum, and step values.
     
    1. Under the "Min" heading, input the minimum value. For example: "400000".
       
    2. Under the "Max" heading, input the maximum value. For example: "2000000".
       
    3. (Optional) Under the "Step" heading, input the step value. For example: "10000".
     
  6. Click the button: "SAVE ATTRIBUTE".
    The "ATTRIBUTE OPTIONS" window will close.
  7. Under the "Value" heading, click on the field. Then input the attribute-value. For example: "770000".
  8. (Optional) If this is the last attribute you're adding, click the button: "SAVE ATTRIBUTES".

True/False

"TRUE/FALSE" only allows a checked or unchecked value in the attribute-value field.
 
In the Map View, the user will click the overlay to view the attribute in the info bubble. For the attribute-value, the user will see either "true" or "false".
  1. In the overlay's "ATTRIBUTES" window, find or add an empty attribute row.
  2. Under the "Shared" heading, click to add a check to the box.
  3. Under the "Attribute" heading, click on the empty field. Then input the attribute-name. For example: "INGROUND POOL".
  4. Under the "Type" heading, click on the drop-down menu. Then select: "TRUE/FALSE".
  5. Under the "Value" heading, click to add a check to the box.
  6. (Optional) If this is the last attribute you're adding, click the button: "SAVE ATTRIBUTES".

Multiple Options

"MULTIPLE OPTIONS" limits the attribute-values to the options you create. Team members select the options from a drop-down menu in the "ATTRIBUTES" window.
 
Team members might find an option's text difficult to read, because of formatting, etc. If you want to make an option easier to read, you can create a human-readable label for the option.
 
For example:
Option = CO
Label = Colorado
 
In the Map View, the user will click the overlay to view the attribute in the info bubble. For the attribute-value, the user will see the option (not the label).
  1. In the overlay's "ATTRIBUTES" window, find or add an empty attribute row.
  2. Under the "Shared" heading, click to add a check to the box.
  3. Under the "Attribute" heading, click on the empty field. Then input the attribute-name. For example: "STATE".
  4. Under the "Type" heading, click on the drop-down menu. Then select: "MULTIPLE OPTIONS".
    The "ATTRIBUTE OPTIONS" window will appear.
  5. Input the option's value, and the label's value.
     
    1. Under the "Option Value" heading, input the option's value. For example: "CO".
       
    2. (Optional) Under the "Label" heading, input the label's value. For example: "Colorado".
     
  6. (Optional) To add another option, click the button: "+ Add Option". Then repeat step #5.
  7. Click the button: "SAVE ATTRIBUTE".
    The "ATTRIBUTES OPTIONS" window will close.
  8. Under the "Value" heading, click on the drop-down menu displaying: "NOT SET". Then select the attribute-value's label. For example: "COLORADO".
  9. (Optional) If this is the last attribute you're adding, click the button: "SAVE ATTRIBUTES".

Auto > Length KM

"LENGTH KM" automatically inputs the line's distance measurement into the attribute-value field. The measurement units are kilometers (KM).
 
You can select the "LENGTH KM" type for: straight lines, freeform lines, precise curved lines, and flight lines.
 
You can also select the "LENGTH KM" type for a shape's border.
  1. In the overlay's "ATTRIBUTES" window, find or add an empty attribute row.
  2. Under the "Shared" heading, click to add a check to the box.
  3. Under the "Attribute" heading, click on the empty field. Then input the attribute-name. For example: "DISTANCE (KM)".
  4. Under the "Type" heading, click on the drop-down menu. Then select: "LENGTH KM".
  5. Under the "Value" heading, you will see the distance measurement is added automatically.
    For example:
    4087.7492
  6. (Optional) If this is the last attribute you're adding, click the button: "SAVE ATTRIBUTES".
