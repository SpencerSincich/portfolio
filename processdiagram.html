Project description



This check-out process is a design document called a User Flow. The main purpose of the user flow is to show the actions a user can perform in a process.



The reading audience is a developer who is building an ecommerce site.



    The document shows:
  • What actions a user can perform in the check-out process.
  • Where details are captured by the database.


    The document will help the developer decide:
  • What template pages to build.
  • How the pages connect together.
  • What options the user will have on each page.
  • Where to connect to the databases.


Project software



I created the diagram by using Microsoft Visio Pro for Office 365. Visio is data visualization software.
